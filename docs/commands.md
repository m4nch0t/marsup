# Hugo commands

- `hugo check`: verifications
- `hugo config`: show configuration
- `hugo convert`: convert elements
- `hugo deploy`: deploy site on cloud provider
- `hugo env`: show environement
- `hugo gen`: giv acces to generators
- `hugo help`: display help on command
- `hugo import`: import site from another place
- `hugo list`: list contents
- `hugo mod`: give acces to help modules
- `hugo new`: create content
- `hugo server`: start local server
- `hugo version`: show version
